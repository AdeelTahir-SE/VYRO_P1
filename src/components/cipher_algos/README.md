🔐 Cipher Algorithms & Required Configuration
Algorithm	Description	Configuration Needed
Caesar Cipher	Shifts each letter by N positions in the alphabet.	Shift amount (integer)
XOR Cipher	XORs each byte of input with a repeating key.	Key (string)
Vigenère Cipher	Polyalphabetic substitution using a keyword.	Keyword (string)
Rail Fence Cipher	Transposition cipher that writes text in a zigzag across N rails.	Number of rails (integer)
Substitution Cipher	Replaces each letter using a custom alphabet mapping.	Substitution alphabet (26 characters)
Columnar Transposition	Rearranges text by writing into rows and reading by column order.	Keyword (determines column order
